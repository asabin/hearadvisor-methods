# HearAdvisor Methods

Public documentation of HearAdvisor's procedures for recording and evaluating hearing devices and earplugs.

**Live site:** [https://asabin.github.io/hearadvisor-methods/](https://asabin.github.io/hearadvisor-methods/)

## About

This repository contains the source for HearAdvisor's methodology documentation. We believe in transparency -- every change to our methodology is tracked here with full version history.

## Contributing

We welcome questions and feedback:

- **Ask a question:** [Open an issue](https://github.com/asabin/hearadvisor-methods/issues/new?template=methodology-question.md)
- **Suggest an improvement:** [Open an issue](https://github.com/asabin/hearadvisor-methods/issues/new?template=feedback.md)

## Building locally

```bash
pip install mkdocs-material
mkdocs serve
```

Then visit `http://localhost:8000`.
