# Metrics

We quantify hearing device performance across five dimensions, each scored on a 0--5 point scale:

| Metric | What it Measures |
|---|---|
| [Speech Perception Benefit](speech-perception.md) | Improvement in speech intelligibility (quiet/moderate and loud) |
| [Occlusion](occlusion.md) | Own-voice quality ("boominess") |
| [Music Streaming Quality](music-streaming.md) | Quality of streamed music via Bluetooth |
| [Feedback](feedback.md) | Resistance to acoustic feedback ("squealing") |
| [SoundScore](soundscore.md) | Weighted composite of all metrics |

These five component scores are combined into a single **SoundScore** using survey-derived importance weights (see [SoundScore](soundscore.md) for details).
