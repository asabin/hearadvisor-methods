# HearAdvisor Methods

This is the official reference for HearAdvisor's testing and evaluation methodology for hearing aids and earplugs. All scores, ratings, and recordings published on [HearAdvisor.com](https://hearadvisor.com) are produced in accordance with the procedures documented here.

This documentation is maintained as a living standard -- every change is publicly tracked in the [changelog](changelog.md). Planned methodology enhancements are published on our [public roadmap](https://github.com/users/asabin/projects/1). We welcome [questions and feedback](https://github.com/asabin/hearadvisor-methods/issues).

## Product Methodologies


|                     | Methodology                                         | Description                                                                                                                                          |
| ------------------- | --------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Hearing Devices** | [View methodology &rarr;](hearing-devices/index.md) | Recording and evaluating hearing aids using KEMAR manikin measurements, acoustic scenes, and validated metrics.                                      |
| **Earplugs**        | [View methodology &rarr;](earplugs/index.md)        | Evaluating earplug loudness reduction and sound quality using KEMAR manikin measurements, perceptual loudness models and spectral distortion models. |
